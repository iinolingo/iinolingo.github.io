Working with Positions

Sticky

This is an example of an element with a sticky position. Its position is based on the user's scroll position.

Basically you set the position of whatever you are working on to "-webkit-sticky" first! Then you set another position property to just "sticky". This will completely set it!

Relative

This is an example of an element with a relative position. It can move around a little, depending on the viewport, but it is in a position relative to itself.

With affecting the values of the top and left properties it actually changed the positive of this relative box.

Fixed

This is an example of an element with a fixed position. Its position is relative to the viewport.

Setting the top to a value of 90 pixels and the right to a value of 10, the fixed box will stay right where it is in the page. It will just essentially follow as you scroll.

Absolute

This is an example of an element with an absolute position. It is similar to the fixed position; however, it is positioned relative to the closest parent element.

I set bottom to 0 and the right property to 0 as well. This made the absolute box stay relatively still at the bottom of the page.
